## AOP vs AspectJ

–ê—Å–ø–µ–∫—Ç–Ω–æ-–æ—Ä–∏–µ–Ω—Ç–∏—Ä–æ–≤–∞–Ω–Ω–æ–µ –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏–µ (AOP) ‚Äî —ç—Ç–æ –ø–∞—Ä–∞–¥–∏–≥–º–∞, –∫–æ—Ç–æ—Ä–∞—è –ø–æ–∑–≤–æ–ª—è–µ—Ç –≤—ã–¥–µ–ª—è—Ç—å —Ç–∞–∫ –Ω–∞–∑—ã–≤–∞–µ–º—É—é —Å–∫–≤–æ–∑–Ω—É—é —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω–æ—Å—Ç—å (cross-cutting concerns) –∏–∑ –æ—Å–Ω–æ–≤–Ω–æ–π –±–∏–∑–Ω–µ—Å-–ª–æ–≥–∏–∫–∏. –ü—Ä–∏–º–µ—Ä—ã —Ç–∞–∫–æ–π —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω–æ—Å—Ç–∏ ‚Äî –ª–æ–≥–∏—Ä–æ–≤–∞–Ω–∏–µ, —É–ø—Ä–∞–≤–ª–µ–Ω–∏–µ —Ç—Ä–∞–Ω–∑–∞–∫—Ü–∏—è–º–∏, –±–µ–∑–æ–ø–∞—Å–Ω–æ—Å—Ç—å, –∫—ç—à–∏—Ä–æ–≤–∞–Ω–∏–µ.

Spring AOP –∏ AspectJ ‚Äî —ç—Ç–æ –¥–≤–µ –ø–æ–ø—É–ª—è—Ä–Ω—ã–µ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ –∫–æ–Ω—Ü–µ–ø—Ü–∏–∏ AOP –≤ –º–∏—Ä–µ Java. –û–Ω–∏ —Ä–µ—à–∞—é—Ç –æ–¥–Ω—É –∏ —Ç—É –∂–µ –∑–∞–¥–∞—á—É, –Ω–æ –¥–µ–ª–∞—é—Ç —ç—Ç–æ –ø–æ-—Ä–∞–∑–Ω–æ–º—É.

### Spring AOP

–û–±–ª–µ–≥—á–µ–Ω–Ω–∞—è —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è AOP, –∫–æ—Ç–æ—Ä–∞—è –ø–æ—Å—Ç–∞–≤–ª—è–µ—Ç—Å—è –≤ –∫–æ–º–ø–ª–µ–∫—Ç–µ —Å —Ñ—Ä–µ–π–º–≤–æ—Ä–∫–æ–º Spring. –û–Ω–∞ –∏–¥–µ–∞–ª—å–Ω–æ –ø–æ–¥—Ö–æ–¥–∏—Ç –¥–ª—è –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–∞ –ø–æ–≤—Å–µ–¥–Ω–µ–≤–Ω—ã—Ö –∑–∞–¥–∞—á. –û—Å–Ω–æ–≤–∞–Ω–∞ –Ω–∞ –ø—Ä–æ–∫—Å–∏-–æ–±—ä–µ–∫—Ç–∞—Ö, –∫–æ—Ç–æ—Ä—ã–µ —Å–æ–∑–¥–∞—é—Ç—Å—è –≤–æ –≤—Ä–µ–º—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è (at runtime).

JDK Dynamic Proxy –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è, –µ—Å–ª–∏ –≤–∞—à –±–∏–Ω —Ä–µ–∞–ª–∏–∑—É–µ—Ç —Ö–æ—Ç—è –±—ã –æ–¥–∏–Ω –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å. Spring —Å–æ–∑–¥–∞–µ—Ç –ø—Ä–æ–∫—Å–∏-–∫–ª–∞—Å—Å, –∫–æ—Ç–æ—Ä—ã–π —Ä–µ–∞–ª–∏–∑—É–µ—Ç —Ç–æ—Ç –∂–µ –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å –∏ –ø–µ—Ä–µ—Ö–≤–∞—Ç—ã–≤–∞–µ—Ç –≤—ã–∑–æ–≤—ã –º–µ—Ç–æ–¥–æ–≤.    

CGLIB –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è, –µ—Å–ª–∏ –±–∏–Ω –Ω–µ —Ä–µ–∞–ª–∏–∑—É–µ—Ç –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å—ã. Spring —Å–æ–∑–¥–∞–µ—Ç –ø–æ–¥–∫–ª–∞—Å—Å –≤–∞—à–µ–≥–æ –∫–ª–∞—Å—Å–∞ –∏ –ø–µ—Ä–µ–æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç –µ–≥–æ –º–µ—Ç–æ–¥—ã –¥–ª—è –ø–µ—Ä–µ—Ö–≤–∞—Ç–∞ –≤—ã–∑–æ–≤–æ–≤.

Spring AOP –º–æ–∂–µ—Ç –ø—Ä–∏–º–µ–Ω—è—Ç—å "—Å–æ–≤–µ—Ç—ã" (advice) —Ç–æ–ª—å–∫–æ –∫ –≤—ã–∑–æ–≤–∞–º –ø—É–±–ª–∏—á–Ω—ã—Ö –º–µ—Ç–æ–¥–æ–≤ Spring-–±–∏–Ω–æ–≤. –û–Ω –Ω–µ –º–æ–∂–µ—Ç –ø–µ—Ä–µ—Ö–≤–∞—Ç—ã–≤–∞—Ç—å:
- –≤—ã–∑–æ–≤—ã –∫–æ–Ω—Å—Ç—Ä—É–∫—Ç–æ—Ä–æ–≤;        
- –¥–æ—Å—Ç—É–ø –∫ –ø–æ–ª—è–º;
- –≤—ã–∑–æ–≤—ã –ø—Ä–∏–≤–∞—Ç–Ω—ã—Ö –∏–ª–∏ final –º–µ—Ç–æ–¥–æ–≤;
- –≤—ã–∑–æ–≤—ã –º–µ—Ç–æ–¥–æ–≤ –≤–Ω—É—Ç—Ä–∏ —Ç–æ–≥–æ –∂–µ –æ–±—ä–µ–∫—Ç–∞ (this.someMethod()), —Ç–∞–∫ –∫–∞–∫ –æ–Ω–∏ –Ω–µ –ø—Ä–æ—Ö–æ–¥—è—Ç —á–µ—Ä–µ–∑ –ø—Ä–æ–∫—Å–∏.

–ü—Ä–µ–∏–º—É—â–µ—Å—Ç–≤–∞:
- –Ω–µ —Ç—Ä–µ–±—É–µ—Ç –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω–æ–π –∫–æ–Ω—Ñ–∏–≥—É—Ä–∞—Ü–∏–∏, –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä–æ–≤ –∏–ª–∏ –∞–≥–µ–Ω—Ç–æ–≤, –≤—Å–µ —Ä–∞–±–æ—Ç–∞–µ—Ç "–∏–∑ –∫–æ—Ä–æ–±–∫–∏" –≤ —Å—Ä–µ–¥–µ Spring;

### AspectJ

–≠—Ç–æ –ø–æ–ª–Ω–æ—Ü–µ–Ω–Ω–∞—è, –º–æ—â–Ω–∞—è –∏ –Ω–µ–∑–∞–≤–∏—Å–∏–º–∞—è —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è AOP. –≠—Ç–æ –º–æ–∂–µ—Ç –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç—å –Ω–∞ —Ä–∞–∑–Ω—ã—Ö —ç—Ç–∞–ø–∞—Ö:
-  –≤–æ –≤—Ä–µ–º—è –∫–æ–º–ø–∏–ª—è—Ü–∏–∏ —Å –ø–æ–º–æ—â—å—é —Å–ø–µ—Ü–∏–∞–ª—å–Ω–æ–≥–æ –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä–∞ ajc (compile-time weaving);

- –≤–ø–ª–µ—Ç–µ–Ω–∏–µ –∫–æ–¥–∞ –≤ –º–æ–º–µ–Ω—Ç –∑–∞–≥—Ä—É–∑–∫–∏ –∫–ª–∞—Å—Å–æ–≤ –≤ JVM —Å –ø–æ–º–æ—â—å—é —Å–ø–µ—Ü–∏–∞–ª—å–Ω–æ–≥–æ Java-–∞–≥–µ–Ω—Ç–∞ (load-time weaving).
        
AspectJ –º–æ–∂–µ—Ç –ø–µ—Ä–µ—Ö–≤–∞—Ç—ã–≤–∞—Ç—å –ø—Ä–∞–∫—Ç–∏—á–µ—Å–∫–∏ –≤—Å—ë:
- –≤—ã–∑–æ–≤—ã –º–µ—Ç–æ–¥–æ–≤ (–ø—É–±–ª–∏—á–Ω—ã—Ö, –ø—Ä–∏–≤–∞—Ç–Ω—ã—Ö, —Å—Ç–∞—Ç–∏—á–µ—Å–∫–∏—Ö);
- –≤—ã–∑–æ–≤—ã –∫–æ–Ω—Å—Ç—Ä—É–∫—Ç–æ—Ä–æ–≤;
- —á—Ç–µ–Ω–∏–µ –∏ –∑–∞–ø–∏—Å—å –ø–æ–ª–µ–π;
- –æ–±—Ä–∞–±–æ—Ç–∫—É –∏—Å–∫–ª—é—á–µ–Ω–∏–π;
- –∏–Ω–∏—Ü–∏–∞–ª–∏–∑–∞—Ü–∏—é –∫–ª–∞—Å—Å–æ–≤.

–ü—Ä–µ–∏–º—É—â–µ—Å—Ç–≤–∞:
